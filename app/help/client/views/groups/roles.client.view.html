Group Roles

All group members receive basic 'read' access to group streams. Advanced access can be granted using the roles described below.

"Group Admin"
This role grants the user the ability to add/remove members of the group and to delete the group itself.
"Editor"
This role grants the user the ability to create/edit/delete resources on behalf of the group.

Creating Groups

There is a special system-level role for users who need the ability to create groups. If you believe you need this type of access, you should contact us directly.
